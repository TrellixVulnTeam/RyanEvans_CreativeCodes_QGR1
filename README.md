## How To Use
From your command line, clone and run VSCode_Template:

```
# Clone this repository
$ git clone https://github.com/YKizou/VSCode_Template

# Go into the repository
$ cd VSCode_Template

# Install dependencies
$ yarn

# Run Project
$ yarn start
```
